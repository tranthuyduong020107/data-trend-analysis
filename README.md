# Data Trend Analysis – Student Project

## Mục tiêu
Phân tích xu hướng dữ liệu và trực quan hóa kết quả để nhận diện các mẫu, biến động và mối quan hệ trong dữ liệu.

## Cấu trúc dự án
- `data/dataset.csv` – dữ liệu sử dụng cho phân tích
- `notebooks/trend_analysis.ipynb` – notebook phân tích chi tiết
- `src/trend_analysis.py` – file Python chạy tự động phân tích và sinh biểu đồ
- `results/figures/` – nơi chứa các biểu đồ xuất ra
